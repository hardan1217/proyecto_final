# WebClimatica
Este proyecto tiene como objetivo ser una pagina web que contenga informacion y poder crear concientizacion y opiniones

#Funciones 
  - un inicio de sesion el cual va a filtrar la informacion de tu primera busqueda para poder filtrar mejor la informacion en tu segunda visita
  - un espacio con informacion sobre el cambio climatico
  - un foro en el cual tu puedes poner tu opinion
  - un metodo para que segun los debates de cada dia en el foro se valla alimentando la pagina de informacion basada en opiniones objetivas

#Tecnologias
  -phyton
  -HTML
  -CSS
